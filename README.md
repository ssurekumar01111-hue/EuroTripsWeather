# 🌦️ EuroTrips Weather  
**Beautiful 7-Day European Weather Forecast App (Premium Frosted-Glass UI)**  
Live Demo: https://euro-weather.netlify.app/

EuroTrips Weather is a modern, premium-looking weather application built using  
**Vanilla JavaScript, HTML, CSS, and Netlify Functions**.  
It provides fast, accurate **7-day European weather forecasts** using the  
7Timer! Forecast API (via a robust JSON-repairing proxy function).

The UI uses a **clean frost-glass design (Theme A)** inspired by iOS/macOS design language.

---

## ✨ Features

### 🌍 **European City Weather Search**
- Enter any major European city (e.g., `Paris,FR`)
- Or click from popular cities like *London, Rome, Berlin, Amsterdam, Vienna*…

### 🌦️ **Accurate 7-Day Forecast**
- Weather condition (text + icon)
- Min / Max temperature
- Precipitation
- Wind forecast

### ❄️ **Premium UI (Theme A — Frosted Glass)**
- Glassmorphism  
- Soft shadows  
- Smooth animations  
- Responsive on all devices  

### 🔥 **Extras**
- °C / °F toggle  
- Recent search history  
- Automatic caching for 30 minutes  
- Faster load time due to local cache  
- Netlify serverless backend to fix malformed 7Timer JSON
